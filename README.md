# SKAMBY- trocas e vendas Sustentáveis

## 📖 Descrição

O SKAMBY é uma plataforma web desenvolvida como uma base de demonstração, com o objetivo de dar nova vida a objetos por meio de um sistema de busca e categorização de produtos.

A ideia do projeto é simular um ambiente onde usuários podem explorar itens como roupas, eletrônicos, móveis e outros, facilitando a reutilização e descoberta de produtos.

## 🚀 Funcionalidades

* Sistema de busca de produtos
* Exibição de produtos recentes
* Navegação por categorias (roupas, eletrônicos, móveis, livros, etc.)
* Interface simples e intuitiva
* Estrutura pronta para expansão (login, perfil, favoritos, chat)

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## ▶️ Como usar

1. Clone o repositório:

```bash
git clone https://github.com/kauanlimatz/skmaby.git
```

2. Acesse a pasta do projeto

3. Abra o arquivo `index.html` no navegador

## 📌 Observações

Este projeto é uma base de demonstração e não possui integração com backend ou banco de dados. Ele foi desenvolvido para fins de estudo e pode ser expandido futuramente.

## 💡 Possíveis melhorias

* Integração com banco de dados
* Sistema de login funcional
* Publicação de produtos pelos usuários
